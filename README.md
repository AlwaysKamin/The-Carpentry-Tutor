# The-Carpenter-Tutor

## Setup
---

Grab all of the dependncies:
  - GLEW
  - GLFW3
  - OpenGL
  - Intel Realsense
  
Clone this repository to wherever you'd like:

```console 
git clone https://github.com/AlwaysKamin/The-Carpentry-Tutor.git 
```

To build:

```console
cd [path]/The-Carpentry-Tutor
mkdir build
cd build
cmake ..
make
```

To Run:

```console
cd [path]/The-Carpentry-Tutor/bin
./[Name of Program]
```
